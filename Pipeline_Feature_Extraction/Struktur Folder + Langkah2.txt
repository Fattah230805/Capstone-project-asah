Petunjuk Menjalankan Pipeline Ekstraksi Fitur

Struktur folder project:

Pipeline_Feature_Extraction/
│
├─ data/
│  ├─ raw_videos/
│  │  └─ video1.webm               # Contoh video yang ingin diproses dan diekstrak fitur
│  ├─ dataset_master/
│  │  └─ labels.csv                # File label untuk training / manual scoring (opsional jika hanya tes video baru)
│  └─ processed/                   # Folder output untuk dataset hasil ekstraksi fitur
│
├─ src/
│  └─ preprocessing/
│      ├─ analyze_audio.py
│      ├─ analyze_stt.py
│      ├─ analyze_visual.py
│      ├─ aggregate_visual.py
│      ├─ merge_audio_visual.py
│      └─ merge_with_labels.py
│
├─ pipeline_features.py
└─ requirements.txt                # Daftar dependensi yang harus diinstall sebelum menjalankan pipeline

Penjelasan:

- Folder data/raw_videos/ berisi video yang ingin kamu olah / ekstrak fiturnya menjadi dataset.
- Folder data/dataset_master/ berisi labels.csv untuk training atau manual scoring. Opsional jika hanya tes video baru.
- Folder data/processed/ akan berisi dataset final hasil ekstraksi fitur.

Cara menjalankan pipeline:

1. Install dependensi terlebih dahulu:

   pip install -r requirements.txt

2. Jalankan pipeline dengan perintah:

   python pipeline_features.py

3. Pipeline akan memproses video di data/raw_videos/, ekstrak fitur, lalu simpan dataset final di data/processed/.

4. Dataset hasil ekstraksi siap digunakan untuk modelling atau testing selanjutnya.
